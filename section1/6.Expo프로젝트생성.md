# 6. Expo 프로젝트 생성하기

## 핵심 요약

- Expo 기반 React Native 프로젝트는 공식 React Native 문서의 Expo 탭 안내를 따라 만들 수 있다.
- 프로젝트 생성 전에 Node.js가 설치되어 있어야 한다.
- Node.js는 브라우저 밖에서 JavaScript를 실행하는 런타임이며, Expo와 `npx` 명령 실행에 필요하다.
- 새 프로젝트는 `npx create-expo-app` 명령으로 만들 수 있다.
- 강의에서는 복잡한 기본 템플릿 대신 `--template blank` 옵션을 사용해 가장 단순한 프로젝트에서 시작한다.

## 학습 목표

- Expo 프로젝트를 생성하기 전에 필요한 준비 사항을 이해한다.
- `npx create-expo-app` 명령어의 역할을 이해한다.
- `--template blank` 옵션을 사용하는 이유를 안다.

## 주요 내용

### Node.js 설치 확인

새로운 Expo 프로젝트를 만들기 전에 먼저 Node.js가 설치되어 있는지 확인해야 한다. Node.js는 브라우저 외부에서 JavaScript를 실행할 수 있게 해주는 JavaScript 런타임이다.

이번 강의에서 직접 Node.js 코드를 작성하는 것은 아니지만, Expo가 내부적으로 Node.js를 사용한다. 또한 새 프로젝트를 생성할 때 실행하는 `npx` 명령도 Node.js 환경이 필요하다.

Node.js가 없다면 `nodejs.org`에서 LTS 버전을 설치하면 된다.

### create-expo-app 명령어

Node.js가 준비되면 터미널이나 명령 프롬프트에서 Expo 기반 React Native 프로젝트를 생성할 수 있다.

먼저 프로젝트를 만들고 싶은 위치로 이동한 다음, 다음과 같은 명령어를 실행한다.

```bash
npx create-expo-app --template blank RNcourse
```

이 명령어는 `RNcourse`라는 이름의 새 폴더를 만들고, 그 안에 Expo 기반 React Native 프로젝트를 설정한다.

### --template blank 옵션

강의에서는 `--template blank` 옵션을 추가한다. 이 옵션을 사용하면 매우 기본적인 프로젝트 구조에서 시작할 수 있다.

옵션을 붙이지 않으면 TypeScript가 포함된 더 복잡한 시작 프로젝트가 생성될 수 있다. 이번 강의는 React Native의 주요 기능을 처음부터 단계별로 배우는 데 집중하므로, 단순한 JavaScript 기반 blank 템플릿으로 시작한다.

### 프로젝트 이름

명령어 마지막에 오는 `RNcourse`는 프로젝트 이름이다. 이 이름은 새로 생성되는 폴더 이름으로도 사용된다.

```bash
npx create-expo-app --template blank 원하는프로젝트이름
```

프로젝트 이름은 자유롭게 정할 수 있다. 강의에서는 React Native Course를 줄여 `RNcourse`라는 이름을 사용한다.

### Expo 공식 사이트

React Native 공식 문서와 별도로 Expo 팀이 제공하는 공식 웹사이트인 `expo.dev`도 있다.

이 사이트에서는 Expo 관련 명령어, Expo가 제공하는 기능, 추가 도구와 서비스에 대해 더 자세히 확인할 수 있다. 다만 강의에서는 필요한 시점마다 Expo 기능을 하나씩 다룬다.

### 프로젝트 생성 후 안내

`create-expo-app` 명령은 프로젝트 구조를 만들고 필요한 의존성을 설치한다. 이 과정은 시간이 조금 걸릴 수 있다.

설치가 끝나면 터미널에 다음 작업을 위한 안내가 표시된다. 보통 새 프로젝트 폴더로 이동하는 명령과 앱을 실행하는 명령이 함께 제공된다.

```bash
cd RNcourse
```

이후 Android, iOS, 웹에서 앱을 실행할 수 있는 명령어가 안내된다. 다만 이 강의에서는 웹 앱이 아니라 Android와 iOS용 React Native 앱 개발에 집중한다.

## 정리

Expo 프로젝트를 만들려면 먼저 Node.js가 설치되어 있어야 하며, 원하는 위치에서 `npx create-expo-app --template blank 프로젝트이름` 명령을 실행하면 된다.

강의에서는 복잡한 템플릿을 피하고 기본 구조부터 이해하기 위해 blank 템플릿을 사용한다. 다음 단계에서는 생성된 프로젝트 폴더를 열고 어떤 파일들이 만들어졌는지 살펴본다.
