# 1. 강의 소개

## 핵심 요약

- 이 강의는 React Native를 기초부터 배우는 과정이다.
- 여러 앱을 직접 만들어보며 필수 기능을 익히는 방식으로 진행된다.
- React Native는 React를 기반으로 하므로 React에 대한 기본 이해가 필요하다.
- 이론 설명과 실습 예시를 함께 다루며, 실제로 다룰 수 있는 수준까지 익히는 것이 목표다.
- 첫 번째 핵심 질문은 "React Native란 무엇인가?"이다.

## 학습 목표

- 강의 전체가 어떤 방식으로 진행되는지 이해한다.
- React Native 학습 전에 React 지식이 필요한 이유를 안다.
- 앞으로 이론과 실습을 함께 따라가며 앱을 만들어볼 준비를 한다.

## 주요 내용

### 강의의 방향

이 강의는 React Native를 처음부터 차근차근 다룬다. 단순히 문법만 설명하는 것이 아니라, 여러 앱을 구축하면서 실제 개발에 필요한 기능을 익히는 데 초점을 둔다.

강의의 목표는 React Native의 개념을 이해하는 것에서 끝나지 않는다. 직접 예제를 따라 만들며 기능을 사용할 수 있는 수준까지 익히는 것이 핵심이다.

### 필요한 사전 지식

React Native는 React를 기반으로 만들어졌기 때문에 React에 대한 기본 지식이 필요하다. 컴포넌트, props, state 같은 React의 핵심 개념을 알고 있으면 강의를 따라가기 훨씬 수월하다.

React를 아직 접해보지 않았다면 별도의 React 입문 강의를 먼저 듣는 것이 좋다. 반드시 특정 강의를 들어야 하는 것은 아니며, React의 기본 원리를 이해하고 있으면 충분하다.

### 학습 방식

강의는 필요한 이론을 먼저 설명하고, 그 이론을 이해할 수 있도록 실질적인 예시를 함께 보여주는 방식으로 진행된다. 이후 여러 앱을 만들면서 React Native의 필수 기능을 자연스럽게 익히게 된다.

첫 강의는 강의 소개로 시작하며, 곧바로 React Native가 무엇인지라는 핵심 질문으로 이어진다.

## 정리

이번 강의는 전체 과정의 방향과 준비 사항을 소개한다. React Native를 제대로 배우려면 React의 기본 개념을 알고 있어야 하며, 앞으로 이론과 실습을 함께 진행하면서 실제 앱 개발 능력을 쌓아가게 된다.
