# 5. Expo로 프로젝트 시작하기

## 핵심 요약

- React Native 프로젝트를 시작할 때는 공식 문서인 `reactnative.dev`를 참고할 수 있다.
- 새 프로젝트를 만드는 대표적인 방법은 Expo CLI와 React Native CLI 두 가지다.
- 강의에서는 더 쉽고 관리가 편한 Expo 방식을 사용한다.
- Expo는 프로젝트 생성, 실행, 빌드, 네이티브 기능 사용을 더 단순하게 만들어준다.
- 필요하다면 나중에 Expo 환경에서 벗어나 React Native CLI 중심의 워크플로로 전환할 수 있다.

## 학습 목표

- React Native 프로젝트를 시작할 때 어떤 도구를 사용할 수 있는지 이해한다.
- Expo CLI와 React Native CLI의 차이를 구분한다.
- 이 강의에서 Expo를 사용하는 이유를 이해한다.

## 주요 내용

### 공식 문서 활용

React Native를 시작할 때는 공식 웹사이트인 `reactnative.dev`를 참고할 수 있다. 이 사이트에는 React Native에 대한 개념 설명, 개발 환경 설정, 프로젝트 생성 방법 등이 정리되어 있다.

공식 문서의 `Get Started` 또는 개발 환경 설정 페이지에서는 새 React Native 프로젝트를 만드는 방법을 확인할 수 있다.

### 프로젝트 생성 도구

React Native 프로젝트를 만들고 실행하기 위한 대표적인 도구는 두 가지다.

```text
Expo CLI
React Native CLI
```

두 도구 모두 React Native 프로젝트 생성, 테스트 기기나 시뮬레이터에서 앱 실행, 앱스토어 배포용 빌드 생성 등을 도와준다.

둘을 동시에 사용할 필요는 없다. 프로젝트를 시작할 때는 둘 중 하나의 워크플로를 선택하면 된다.

### Expo를 사용하는 이유

강의에서는 Expo CLI를 사용한다. Expo는 React Native 앱을 더 쉽게 만들고 관리할 수 있게 해주는 도구와 서비스의 모음이다.

Expo를 사용하면 프로젝트 생성이 간단해지고, 앱 실행과 개발 과정도 더 부드러워진다. 카메라처럼 네이티브 기기 기능을 사용할 때도 Expo에서 제공하는 패키지와 도구 덕분에 직접 설정해야 할 일이 줄어든다.

또한 Expo는 기본적으로 무료로 사용할 수 있다. 추가 유료 서비스가 있기는 하지만, React Native 앱을 만들고 배포하는 기본적인 과정에 반드시 비용이 필요한 것은 아니다.

### 관리형 앱 개발 워크플로

Expo는 강의에서 말하는 관리형 앱 개발 워크플로를 제공한다. 이는 프로젝트 생성, 코드 작성, 앱 실행, 네이티브 기능 활용 같은 작업을 더 단순하게 처리할 수 있다는 의미다.

처음 React Native를 배우는 입장에서는 복잡한 네이티브 설정에 시간을 많이 쓰기보다, React Native의 핵심 개념과 앱 개발 흐름에 집중할 수 있는 장점이 있다.

### React Native CLI와의 차이

React Native CLI는 React Native 팀과 커뮤니티에서 제공하는 기본 개발 도구다. Expo보다 더 직접적인 React Native 개발 환경을 제공하지만, 그만큼 설정과 구성을 개발자가 직접 처리해야 하는 경우가 많다.

특히 카메라 같은 네이티브 기능을 사용하거나 플랫폼별 설정을 다룰 때 Expo보다 더 많은 작업이 필요할 수 있다.

반면 React Native CLI는 Java, Kotlin, Swift, Objective-C 같은 네이티브 소스 코드와 더 직접적으로 통합하기 쉽다는 장점이 있다. JavaScript 코드와 네이티브 코드를 깊게 연결해야 하는 프로젝트라면 React Native CLI가 더 적합할 수 있다.

### 나중에 전환할 수 있다는 장점

Expo로 시작한다고 해서 반드시 Expo에만 묶이는 것은 아니다. 필요해지면 Expo 중심의 워크플로에서 벗어나 React Native CLI 방식으로 전환할 수 있다.

이 때문에 처음에는 Expo로 쉽게 시작하고, 나중에 더 낮은 수준의 네이티브 제어가 필요해졌을 때 전환하는 전략이 가능하다.

## 정리

React Native 프로젝트를 시작하는 방법에는 Expo CLI와 React Native CLI가 있다. 이 강의에서는 더 쉽고 빠르게 시작할 수 있으며, 필요할 때 전환도 가능한 Expo를 사용한다.

Expo는 React Native 학습 초반에 복잡한 설정 부담을 줄이고, 앱 개발 자체에 집중할 수 있게 도와주는 도구다.
