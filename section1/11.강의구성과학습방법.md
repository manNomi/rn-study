# 11. 강의 구성과 학습 방법

## 핵심 요약

- 시작 섹션 이후에는 React Native의 핵심 개념을 실제 앱 제작과 함께 배운다.
- 사용자 인터페이스, 스타일링, 레이아웃, 내비게이션 같은 기본기를 먼저 다룬다.
- 이후 상태 관리, 사용자 입력, HTTP 요청, 인증, 네이티브 기능, 푸시 알림, 앱스토어 배포 같은 심화 주제로 확장한다.
- 강의를 제대로 활용하려면 영상을 보기만 하기보다 직접 코드를 작성하고 실험하는 것이 중요하다.
- 막히는 부분이 있으면 강의 리소스, 코드 스냅샷, Q&A 섹션을 적극적으로 활용한다.

## 학습 목표

- 앞으로 강의가 어떤 순서로 진행되는지 이해한다.
- React Native 학습 전에 필요한 JavaScript와 React 지식의 중요성을 확인한다.
- VOD 강의를 효과적으로 활용하는 방법을 정리한다.

## 주요 내용

### 강의의 큰 흐름

초반 시작 섹션을 마친 뒤에는 React Native의 핵심 개념으로 넘어간다. 다음 섹션부터는 기본적인 React Native 컴포넌트를 사용해 사용자 인터페이스를 만들고, 스타일과 레이아웃을 구성하는 방법을 실제 앱을 만들며 익힌다.

이후에는 여러 화면을 오가는 내비게이션을 추가한다. 화면 전환, 애니메이션, 드로어 메뉴, 탭 내비게이션 같은 앱에서 자주 쓰이는 구조를 다루게 된다.

### 필요한 사전 지식

강의 내용을 제대로 이해하려면 JavaScript와 React.js에 대한 기본 지식이 필요하다.

강의 후반에는 JavaScript와 React 복습 섹션도 제공되지만, 짧은 복습 영상이 전체 입문 강의를 완전히 대체할 수는 없다. React 지식이 흐릿하다면 React Native 본편을 시작하기 전에 복습 섹션을 먼저 듣는 것이 도움이 된다.

### 심화 주제

React Native의 핵심을 배운 뒤에는 더 현실적인 앱 개발 주제로 넘어간다.

```text
Context API
Redux
사용자 입력 처리
HTTP 요청
사용자 인증
기기 저장소
카메라
사용자 위치
지도
공유 기능
푸시 알림
앱스토어 배포
```

이 과정에서는 앱을 처음부터 끝까지 완성하면서 앞에서 배운 내용을 반복해서 적용한다. 단순히 개념만 보는 것이 아니라 실제 앱 기능으로 연결하는 것이 중요하다.

### 영상을 활용하는 방법

VOD 강의의 장점은 자신의 속도에 맞춰 학습할 수 있다는 점이다. 필요한 경우 재생 속도를 조절하고, 이해가 잘 되지 않는 부분은 되돌려 다시 보는 것이 좋다.

한 번에 완전히 이해되지 않는 강의나 섹션은 반복해서 들어도 된다. 특히 새로운 개념이 많을 때는 잠시 멈추고 내용을 정리하거나, 직접 코드를 따라 작성해보는 과정이 도움이 된다.

### 직접 코딩하기

강의를 보기만 하는 것보다 직접 코드를 작성하며 따라 하는 편이 훨씬 효과적이다. 강사가 작성하는 코드를 그대로 따라 해도 좋고, 중간에 잠시 멈춘 뒤 스스로 구현해보는 것도 좋다.

익숙해지면 강의에서 만든 앱에 자신만의 작은 기능을 추가해볼 수 있다. 이렇게 변형하고 실험하는 과정에서 React Native의 개념을 더 깊게 이해할 수 있다.

### 리소스와 코드 스냅샷 활용

코딩을 따라 하다 보면 예상과 다르게 동작하거나 에러가 발생할 수 있다. 이때 강의에 첨부된 슬라이드, 코드 스냅샷, 리소스를 활용해 자신의 코드와 비교할 수 있다.

필요하다면 강사의 코드로 일부를 대체하거나, 차이점을 확인하면서 에러 원인을 찾아볼 수 있다. 다음 강의에서는 이런 리소스를 다운로드하고 활용하는 방법을 다룬다.

### Q&A 섹션 활용

질문이 생기면 Q&A 섹션을 활용할 수 있다. 질문을 올리는 것도 좋지만, 다른 수강생의 질문을 읽고 답변을 시도해보는 것도 좋은 학습 방법이다.

다른 사람의 문제를 해결하려면 배운 내용을 명확히 이해하고 설명할 수 있어야 한다. 따라서 Q&A에 참여하는 과정 자체가 강의 내용을 더 깊게 익히는 데 도움이 된다.

## 정리

이 강의는 React Native의 기본 컴포넌트와 스타일링에서 시작해, 내비게이션, 상태 관리, 네이티브 기능, 인증, 푸시 알림, 앱 배포까지 점진적으로 확장된다.

가장 중요한 학습 방법은 영상을 수동적으로 보기보다 직접 코드를 작성하고, 막히는 부분은 리소스와 Q&A를 활용해 해결해보는 것이다. 다음 섹션부터는 React Native의 기본기를 본격적으로 다룬다.
