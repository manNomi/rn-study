# 8. Expo Go로 앱 미리보기

## 핵심 요약

- React Native 앱을 개발할 때는 변경 사항을 실제 기기에서 계속 미리보며 작업하는 흐름이 중요하다.
- Expo를 사용하면 Expo Go 앱을 통해 Android 기기나 iPhone에서 쉽게 프로젝트를 실행해볼 수 있다.
- 프로젝트 폴더에서 `npm start`를 실행하면 Expo 개발 서버가 시작된다.
- 개발 서버는 QR 코드와 로그를 제공하며, Expo Go 또는 기기 카메라로 QR 코드를 스캔해 앱을 열 수 있다.
- `App.js`를 수정하고 저장하면 실제 기기의 미리보기 화면에 변경 사항이 반영된다.

## 학습 목표

- Expo Go 앱을 이용해 Expo 기반 React Native 프로젝트를 실제 기기에서 실행하는 흐름을 이해한다.
- `npm start`로 Expo 개발 서버를 실행하는 이유를 안다.
- 코드 수정 후 저장하면 미리보기에 반영되는 개발 워크플로를 이해한다.

## 주요 내용

### 앱 미리보기의 필요성

React Native 앱을 만들 때는 코드를 수정한 뒤 실제 화면에서 어떻게 보이는지 계속 확인해야 한다. React 웹 앱을 브라우저에서 확인하며 개발하는 것처럼, React Native 앱도 미리보기 환경이 있으면 훨씬 편하게 작업할 수 있다.

Expo를 사용하면 별도의 복잡한 설정 없이 실제 Android 기기나 iPhone에서 앱을 빠르게 미리볼 수 있다.

### Expo Go 설치

실제 기기에서 Expo 앱을 실행하려면 먼저 Expo Go 앱을 설치해야 한다.

```text
Android: Google Play에서 Expo Go 설치
iOS: App Store에서 Expo Go 설치
```

Expo Go는 Expo 팀에서 제공하는 앱이며, 로컬 컴퓨터에서 개발 중인 Expo 기반 React Native 앱을 실제 기기에서 실행할 수 있게 해준다.

### Expo 개발 서버 실행

프로젝트를 미리보기하려면 터미널에서 Expo 개발 서버를 실행해야 한다. VS Code를 사용한다면 `Terminal` 메뉴에서 `New Terminal`을 열어 프로젝트 폴더 안에서 명령어를 실행할 수 있다.

```bash
npm start
```

이 명령은 Expo 개발 서버를 시작한다. 개발 중에는 이 프로세스를 계속 실행한 상태로 두어야 한다.

Expo 개발 서버는 프로젝트 코드를 읽고 JavaScript 번들을 만들어 미리보기할 수 있도록 준비한다. 실행 후 터미널이나 브라우저 화면에서 QR 코드와 로그 메시지를 확인할 수 있다.

### QR 코드로 기기 연결하기

Android에서는 Expo Go 앱 안의 QR 코드 스캐너를 사용해 터미널이나 브라우저에 표시된 QR 코드를 스캔할 수 있다.

iPhone에서는 Expo Go 앱 안에 스캐너가 따로 있는 방식이 아니라, 기본 카메라 앱으로 QR 코드를 스캔한다. QR 코드가 제공하는 링크를 열면 Expo Go 앱이 실행된다.

처음 실행할 때 권한 요청이 나타날 수 있으며, 필요한 권한을 승인하면 앱이 로드된다. 아무 반응이 없다면 QR 코드를 다시 스캔해볼 수 있다.

### JavaScript 번들 로딩

QR 코드로 연결하면 Expo Go 앱이 로컬 개발 서버에 접속하고 JavaScript 번들을 로드한다.

이 번들은 현재 개발 중인 React Native 앱 코드이며, 로딩이 끝나면 실제 기기에서 앱 화면을 확인할 수 있다.

### 코드 변경 확인

앱이 올바르게 연결되었는지 확인하려면 `App.js`의 텍스트를 바꿔볼 수 있다.

예를 들어 화면에 표시되는 문구를 `Hello World!`로 바꾸고 파일을 저장하면, Expo Go에서 실행 중인 앱 화면에도 변경 사항이 반영된다.

이 흐름을 통해 코드를 수정하고, 저장하고, 실제 기기에서 결과를 확인하는 개발 사이클을 반복할 수 있다.

### Expo 워크플로의 장점

Expo Go를 사용하면 실제 기기에 앱을 설치하거나 복잡한 네이티브 빌드 과정을 거치지 않아도 개발 중인 앱을 바로 확인할 수 있다.

이 때문에 Expo는 React Native 앱을 처음 만들고 테스트하는 과정을 훨씬 수월하게 해준다. React Native 자체를 대체하는 것은 아니지만, 개발과 테스트 흐름을 편하게 만들어주는 도구다.

## 정리

Expo 기반 React Native 프로젝트는 `npm start`로 개발 서버를 실행하고, Expo Go 앱으로 QR 코드를 스캔해 실제 기기에서 미리볼 수 있다.

이제 코드를 수정하고 저장할 때마다 기기에서 결과를 확인할 수 있으므로, 본격적으로 React Native 앱을 개발할 준비가 된다.
