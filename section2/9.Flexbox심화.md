# 9. Flexbox 심화

## 핵심 요약

- React Native의 모든 `View`는 기본적으로 Flexbox 레이아웃을 사용한다.
- React Native의 기본 `flexDirection`은 `column`이며, 자식 요소는 위에서 아래로 배치된다.
- `flexDirection`은 주축(Main Axis)의 방향을 결정한다.
- `justifyContent`는 주축을 기준으로 자식 요소를 정렬한다.
- `alignItems`는 교차축(Cross Axis)을 기준으로 자식 요소를 정렬한다.
- 자식 요소의 `flex` 값은 주축을 따라 사용 가능한 공간을 어떤 비율로 나눌지 결정한다.

## 학습 목표

- React Native Flexbox의 기본값을 이해한다.
- 주축과 교차축의 개념을 명확히 구분한다.
- `flexDirection`, `justifyContent`, `alignItems`의 역할을 정리한다.
- 자식 요소에 적용하는 `flex` 값이 공간 배분에 어떤 영향을 주는지 이해한다.

## 주요 내용

### React Native의 View는 기본적으로 Flexbox다

웹에서는 `div`가 기본적으로 Flexbox 컨테이너가 아니다. CSS에서 `display: flex`를 설정해야 Flexbox가 적용된다.

React Native에서는 다르다. `View`는 기본적으로 Flexbox를 사용해 자식 요소를 배치한다. 따라서 별도의 설정 없이도 `flexDirection`, `justifyContent`, `alignItems`, `flex` 같은 Flexbox 관련 스타일을 바로 사용할 수 있다.

### 기본 배치 방향

React Native에서 `flexDirection`의 기본값은 `column`이다.

```text
column: 위에서 아래로 배치
row: 왼쪽에서 오른쪽으로 배치
row-reverse: 오른쪽에서 왼쪽으로 배치
column-reverse: 아래에서 위로 배치
```

따라서 `View` 안에 박스 세 개를 넣으면 기본적으로 세로 방향으로 쌓인다.

```jsx
<View>
  <View><Text>1</Text></View>
  <View><Text>2</Text></View>
  <View><Text>3</Text></View>
</View>
```

가로로 배치하려면 부모 `View`에 `flexDirection: 'row'`를 설정한다.

```jsx
const styles = StyleSheet.create({
  container: {
    flexDirection: 'row',
  },
});
```

### 주축과 교차축

Flexbox에서는 두 축을 기준으로 요소를 배치한다.

```text
Main Axis: flexDirection이 정하는 주된 배치 방향
Cross Axis: Main Axis와 수직인 방향
```

`flexDirection`이 `row`이면 주축은 왼쪽에서 오른쪽으로 흐르는 가로축이다. 이때 교차축은 위에서 아래로 흐르는 세로축이다.

`flexDirection`이 `column`이면 주축은 위에서 아래로 흐르는 세로축이다. 이때 교차축은 왼쪽에서 오른쪽으로 흐르는 가로축이다.

반전 방향도 축의 방향을 바꾼다.

```text
row-reverse: 주축이 오른쪽에서 왼쪽
column-reverse: 주축이 아래에서 위
```

### justifyContent

`justifyContent`는 주축을 기준으로 자식 요소를 어떻게 배치할지 결정한다.

```jsx
const styles = StyleSheet.create({
  container: {
    flexDirection: 'row',
    justifyContent: 'space-between',
  },
});
```

`flexDirection`이 `row`라면 `justifyContent`는 가로 방향의 배치를 제어한다.

자주 사용하는 값은 다음과 같다.

```text
flex-start: 주축의 시작 지점에 배치
center: 주축의 중앙에 배치
flex-end: 주축의 끝 지점에 배치
space-between: 요소 사이에 가능한 공간을 분배
space-around: 요소 주변에 공간을 분배
```

### alignItems

`alignItems`는 교차축을 기준으로 자식 요소를 어떻게 정렬할지 결정한다.

```jsx
const styles = StyleSheet.create({
  container: {
    flexDirection: 'row',
    justifyContent: 'space-between',
    alignItems: 'center',
  },
});
```

`flexDirection`이 `row`라면 교차축은 세로 방향이므로, `alignItems`는 자식 요소를 위아래 방향에서 어떻게 정렬할지 제어한다.

기본값은 `stretch`다. 이 값은 자식 요소가 교차축 방향으로 가능한 공간을 채우도록 늘린다.

```text
stretch: 교차축 방향으로 늘림
center: 교차축 중앙에 배치
flex-start: 교차축 시작 지점에 배치
flex-end: 교차축 끝 지점에 배치
```

### 부모 크기와 자식 크기

부모 `View`에 `width`와 `height`를 설정하면, Flexbox 컨테이너의 사용 가능한 공간이 정해진다.

```jsx
const styles = StyleSheet.create({
  container: {
    width: '80%',
    height: 300,
    flexDirection: 'row',
  },
});
```

이때 자식 요소에 고정된 `width`와 `height`가 없다면, 자식 요소는 기본적으로 자신의 콘텐츠를 표시하는 데 필요한 최소 크기만 차지한다.

다만 교차축 방향에서는 부모의 `alignItems` 기본값인 `stretch` 때문에 자식 요소가 늘어날 수 있다.

### 자식 요소의 flex

자식 요소에 `flex`를 설정하면 주축 방향에서 사용 가능한 공간을 차지한다.

```jsx
const styles = StyleSheet.create({
  redBox: {
    flex: 1,
  },
});
```

`flex: 1`을 가진 자식 요소는 주축 방향으로 가능한 공간을 최대한 차지한다. 다른 자식 요소에 `flex`가 없다면, 그 요소들은 자신의 콘텐츠에 필요한 공간만 차지하고 나머지 공간을 `flex`가 있는 요소가 가져간다.

여러 자식 요소가 `flex` 값을 가지면, 숫자의 비율에 따라 공간이 나뉜다.

```jsx
const styles = StyleSheet.create({
  redBox: {
    flex: 1,
  },
  blueBox: {
    flex: 2,
  },
});
```

위 예시에서는 사용 가능한 공간을 총 `1 + 2 = 3`의 비율로 나눈다. 빨간 박스는 3분의 1, 파란 박스는 3분의 2를 차지한다.

`flex` 값은 절대 크기가 아니라 같은 Flexbox 컨테이너 안에서 서로 비교되는 상대적인 값이다.

### 중첩된 View와 Flexbox

앱 전체에서 Flexbox를 사용하는 `View`가 하나만 있어야 하는 것은 아니다. 여러 `View`를 중첩해서 각각의 영역마다 다른 Flexbox 설정을 적용할 수 있다.

예를 들어 바깥쪽 `View`에서는 박스들을 가로로 배치하고, 각 박스 내부의 `View`에서는 숫자를 중앙에 배치할 수 있다.

```jsx
const styles = StyleSheet.create({
  box: {
    justifyContent: 'center',
    alignItems: 'center',
  },
});
```

이 경우 각 박스 내부에서도 Flexbox가 동작하므로, `justifyContent`와 `alignItems`로 텍스트를 수평과 수직 중앙에 배치할 수 있다.

## 정리

Flexbox는 React Native에서 콘텐츠를 조직하고 배치하는 핵심 도구다. 부모 `View`에서는 `flexDirection`, `justifyContent`, `alignItems`로 자식 요소의 방향과 정렬을 제어하고, 자식 요소에서는 `flex`로 사용 가능한 공간을 어떤 비율로 차지할지 정한다.

React Native 앱에서는 여러 `View`를 중첩하고 각 `View`에 Flexbox 스타일을 적용하면서 원하는 화면 구조를 만들어간다.
