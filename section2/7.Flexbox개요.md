# 7. Flexbox 개요

## 핵심 요약

- React Native에서 레이아웃을 구성할 때 Flexbox는 핵심 개념이다.
- Flexbox는 컨테이너 내부의 자식 요소를 어떻게 배치할지 제어하는 스타일 프로퍼티들의 집합이다.
- React Native의 Flexbox는 CSS Flexbox와 매우 비슷하지만, React Native 스타일 프로퍼티로 설정한다.
- `flexDirection`은 주요 축(Main Axis)이 세로인지 가로인지 결정한다.
- `justifyContent`와 `alignItems`는 각 축을 기준으로 자식 요소의 위치와 간격을 조정한다.

## 학습 목표

- React Native 레이아웃에서 Flexbox가 중요한 이유를 이해한다.
- Flexbox에서 컨테이너와 자식 요소의 관계를 이해한다.
- 주요 축(Main Axis)과 교차 축(Cross Axis)의 개념을 파악한다.
- `flex`, `flexDirection`, `justifyContent`, `alignItems`의 기본 역할을 구분한다.

## 주요 내용

### Flexbox가 필요한 이유

React Native 앱은 다양한 화면 크기와 기기에서 동작한다. 따라서 요소를 고정된 위치에만 배치하기보다, 화면 공간에 맞게 유연하게 배치할 수 있어야 한다.

Flexbox는 이런 레이아웃을 만들기 위한 핵심 도구다. 컨테이너 안에 있는 요소들을 가로 또는 세로로 배치하고, 요소 사이의 간격과 정렬 방식을 제어할 수 있다.

### 컨테이너와 자식 요소

Flexbox는 보통 자식 요소를 담고 있는 컨테이너에 스타일을 적용해 동작한다.

```jsx
const styles = StyleSheet.create({
  container: {
    flex: 1,
    flexDirection: 'column',
    justifyContent: 'center',
    alignItems: 'center',
  },
});
```

위 스타일은 컨테이너가 차지할 공간과, 그 안의 자식 요소들이 어떤 방향과 정렬 방식으로 배치될지를 결정한다.

### flex

`flex`는 요소가 사용 가능한 공간을 얼마나 차지할지 결정한다.

```jsx
flex: 1
```

컨테이너에 `flex: 1`을 설정하면, 해당 컨테이너는 부모 안에서 사용할 수 있는 공간을 최대한 차지하려고 한다. 다만 다른 형제 컨테이너가 있다면 그 요소들과 공간을 나누게 된다.

### flexDirection

`flexDirection`은 자식 요소가 배치되는 방향을 결정한다. 이 방향이 Flexbox의 주요 축(Main Axis)이 된다.

```text
column: 세로 방향 배치
row: 가로 방향 배치
```

React Native에서는 기본값이 `column`이다. 즉 별도로 설정하지 않으면 자식 요소는 위에서 아래로 배치된다.

```jsx
flexDirection: 'column'
```

가로로 배치하려면 `row`를 사용한다.

```jsx
flexDirection: 'row'
```

### 주요 축과 교차 축

Flexbox에서는 두 축이 중요하다.

```text
Main Axis: flexDirection으로 정해지는 주된 배치 방향
Cross Axis: Main Axis와 수직인 방향
```

`flexDirection`이 `column`이면 주요 축은 세로 방향이고, 교차 축은 가로 방향이다.

`flexDirection`이 `row`이면 주요 축은 가로 방향이고, 교차 축은 세로 방향이다.

### justifyContent와 alignItems

`justifyContent`는 주요 축을 기준으로 자식 요소를 어떻게 배치할지 결정한다.

```jsx
justifyContent: 'center'
```

예를 들어 주요 축이 세로 방향이면, `justifyContent`는 자식 요소를 세로 방향으로 어떻게 배치할지 제어한다.

`alignItems`는 교차 축을 기준으로 자식 요소를 어떻게 정렬할지 결정한다.

```jsx
alignItems: 'center'
```

예를 들어 주요 축이 세로 방향인 `column`에서는 교차 축이 가로 방향이므로, `alignItems`는 자식 요소의 가로 정렬을 제어한다.

## 정리

Flexbox는 React Native에서 화면 레이아웃을 구성하는 핵심 방식이다. 컨테이너에 `flex`, `flexDirection`, `justifyContent`, `alignItems` 같은 스타일을 적용해 자식 요소의 방향, 크기, 정렬, 간격을 제어한다.

React Native의 UI를 제대로 구성하려면 Flexbox의 주요 축과 교차 축 개념을 이해하는 것이 중요하다.
