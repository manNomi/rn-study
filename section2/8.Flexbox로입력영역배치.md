# 8. Flexbox로 입력 영역 배치

## 핵심 요약

- `TextInput`과 `Button`을 담고 있는 `View`에 Flexbox 스타일을 적용해 입력 영역을 정렬한다.
- React Native의 `View`는 기본적으로 Flexbox가 활성화되어 있으며, 기본 `flexDirection`은 `column`이다.
- `flexDirection: 'row'`를 적용하면 자식 요소가 가로 방향으로 배치된다.
- `justifyContent: 'space-between'`을 사용하면 주축 방향에서 요소 사이의 공간을 조정할 수 있다.
- `TextInput`에는 `width`, `borderWidth`, `borderColor`, `marginRight`, `padding` 같은 스타일을 적용해 입력창 형태를 만든다.

## 학습 목표

- Flexbox를 사용해 입력창과 버튼을 가로로 배치하는 방법을 이해한다.
- `flexDirection`의 기본값과 `row` 설정의 차이를 파악한다.
- 퍼센트 기반 `width`를 React Native 스타일에서 사용하는 방법을 익힌다.
- 입력 요소에 테두리와 여백을 적용하는 기본 스타일링을 복습한다.

## 주요 내용

### 입력 영역 컨테이너에 스타일 적용

목표 입력 영역은 `TextInput`과 `Button`을 담고 있는 `View`다. 이 `View`에 스타일을 적용해 두 요소의 배치를 조정한다.

먼저 `StyleSheet`에 `inputContainer` 스타일을 정의한다.

```jsx
const styles = StyleSheet.create({
  inputContainer: {
    flexDirection: 'row',
  },
});
```

그리고 이 스타일을 입력 영역의 `View`에 적용한다.

```jsx
<View style={styles.inputContainer}>
  <TextInput placeholder="Your course goal!" />
  <Button title="Add Goal" />
</View>
```

### flexDirection

React Native의 `View`는 기본적으로 Flexbox 레이아웃을 사용한다. 기본 `flexDirection`은 `column`이기 때문에 자식 요소가 위에서 아래로 배치된다.

```text
기본값 column: 세로 배치
row: 가로 배치
```

`flexDirection: 'row'`를 설정하면 `TextInput`과 `Button`이 같은 줄에 나란히 배치된다.

### justifyContent

`justifyContent`는 Flexbox의 주축(Main Axis)을 기준으로 자식 요소를 어떻게 배치할지 결정한다.

`flexDirection`이 `row`인 경우 주축은 가로 방향이다. 따라서 `justifyContent`는 가로 방향에서 요소 사이의 간격을 조정한다.

```jsx
const styles = StyleSheet.create({
  inputContainer: {
    flexDirection: 'row',
    justifyContent: 'space-between',
  },
});
```

`space-between`은 첫 요소와 마지막 요소를 양끝에 가깝게 배치하고, 요소 사이의 공간을 벌린다.

### TextInput 스타일링

입력창이 더 잘 보이도록 `TextInput`에 별도 스타일을 적용한다.

```jsx
const styles = StyleSheet.create({
  textInput: {
    borderWidth: 1,
    borderColor: '#cccccc',
    width: '80%',
  },
});
```

`borderWidth`는 테두리 두께를 숫자로 설정하고, `borderColor`는 색상 문자열을 사용한다.

`width`는 숫자로 픽셀 값을 줄 수도 있지만, 퍼센트 값을 사용할 때는 문자열로 작성해야 한다.

```jsx
width: '80%'
```

이 설정은 `TextInput`이 자신을 담고 있는 컨테이너 너비의 80%를 차지하게 만든다.

### 여백과 안쪽 여백 추가

입력창이 버튼과 너무 붙어 보이면 오른쪽 바깥 여백을 추가할 수 있다.

```jsx
marginRight: 8
```

또 입력한 텍스트가 입력창 테두리에 너무 붙지 않도록 안쪽 여백을 추가한다.

```jsx
padding: 8
```

최종적으로 입력창 스타일은 다음처럼 정리할 수 있다.

```jsx
const styles = StyleSheet.create({
  textInput: {
    borderWidth: 1,
    borderColor: '#cccccc',
    width: '80%',
    marginRight: 8,
    padding: 8,
  },
});
```

그리고 `TextInput`에 스타일을 적용한다.

```jsx
<TextInput
  style={styles.textInput}
  placeholder="Your course goal!"
/>
```

### 아직 남은 개선점

입력창과 버튼이 가로로 배치되고 입력창 스타일도 적용되었지만, 레이아웃이 완성된 것은 아니다.

왼쪽과 오른쪽 간격이 완전히 균형 잡힌 것은 아니고, Android 기본 버튼의 모습도 아직 원하는 디자인과 다를 수 있다. 이런 부분은 이후 스타일링과 레이아웃을 더 다루면서 개선한다.

## 정리

이번 강의에서는 Flexbox를 사용해 `TextInput`과 `Button`을 가로로 배치했다. `flexDirection: 'row'`로 배치 방향을 바꾸고, `TextInput`에 너비와 여백, 테두리, 안쪽 여백을 적용해 입력 영역의 기본 형태를 만들었다.

다음 단계에서는 Flexbox의 다양한 속성을 더 깊게 살펴보며 레이아웃을 정교하게 다듬는다.
