# 1. 섹션 소개

## 핵심 요약

- 이번 섹션에서는 React Native의 핵심 개념과 기본기를 다룬다.
- 핵심 컴포넌트를 사용해 사용자 인터페이스를 구축하는 방법을 배운다.
- React Native 앱과 컴포넌트의 스타일을 변경하는 방법을 익힌다.
- 버튼 클릭 같은 사용자 상호작용을 추가하고 상태를 관리하는 방법을 배운다.
- 실습 앱으로 목표를 기록하고 관리하는 간단한 React Native 앱을 만든다.

## 학습 목표

- React Native의 기본 컴포넌트가 어떤 역할을 하는지 이해한다.
- 컴포넌트를 조합해 화면을 구성하는 흐름을 익힌다.
- 스타일링과 레이아웃을 적용해 원하는 UI를 만드는 방법을 배운다.
- 사용자 입력과 상태 관리를 통해 앱에 상호작용성을 추가한다.

## 주요 내용

### React Native 기본기

이번 섹션은 React Native를 본격적으로 배우는 첫 단계다. 핵심 컴포넌트를 살펴보고, 이 컴포넌트들을 사용해 실제 사용자 인터페이스를 구성한다.

웹에서 HTML 요소를 조합해 화면을 만들듯이, React Native에서는 React Native가 제공하는 컴포넌트를 조합해 모바일 앱 화면을 만든다.

### 스타일링과 레이아웃

React Native 앱은 컴포넌트에 스타일을 적용해 원하는 모양으로 만들 수 있다. 이번 섹션에서는 앱 전체와 개별 컴포넌트의 스타일을 바꾸고, 효과적인 레이아웃을 구성하는 방법을 배운다.

스타일링 방식은 웹 CSS와 비슷한 부분도 있지만 React Native만의 차이도 있으므로, 실습을 통해 자연스럽게 익히게 된다.

### 상호작용과 상태 관리

앱은 단순히 화면을 보여주는 것에서 끝나지 않는다. 사용자가 버튼을 누르거나 입력을 하면 그에 따라 화면이 바뀌어야 한다.

이번 섹션에서는 버튼 클릭 같은 상호작용을 추가하고, React의 상태(State)를 사용해 앱 데이터를 관리하는 방법을 배운다.

### 실습 앱

이번 섹션에서 만들 앱은 목표를 기록할 수 있는 간단한 React Native 앱이다.

이 앱에서는 다음 기능을 구현한다.

```text
목표 목록 표시
모달 오버레이로 새 목표 입력
목표 추가
목표 터치 시 삭제
```

앱 자체는 복잡하지 않지만, React Native의 핵심 개념을 배우기에 적합한 구조를 가지고 있다.

## 정리

이번 섹션에서는 React Native의 기본 컴포넌트, 스타일링, 레이아웃, 상호작용, 상태 관리를 실제 목표 관리 앱을 만들며 익힌다.

작은 앱을 완성하는 과정에서 React Native 앱 개발의 기본 흐름을 자연스럽게 경험하게 된다.
