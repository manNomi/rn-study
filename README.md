# React Native Lecture Notes

React Native 강의 스크립트를 한국어 마크다운 학습 노트로 정리해 GitHub에 보관하는 저장소입니다.

이 저장소의 목적은 강의 대본을 그대로 옮기는 것이 아니라, 강의에서 다루는 핵심 개념과 흐름을 다시 정리해 나중에 복습하기 쉬운 자료로 만드는 것입니다.

## 정리 방식

1. 한국어 강의 스크립트를 입력으로 받습니다.
2. 강의 번호를 기준으로 섹션 폴더와 파일명을 정합니다.
3. 반복 표현, 인사말, 불필요한 구어체는 제거합니다.
4. 핵심 개념, 학습 목표, 중요한 설명, 다음 강의와의 연결점을 중심으로 요약합니다.
5. 결과물은 GitHub에서 바로 읽기 좋은 마크다운 문서로 저장합니다.

## 파일 경로 규칙

강의 번호가 `N-M` 형식이면 아래처럼 저장합니다.

```text
sectionN/M.강의제목.md
```

예시:

```text
1-1 강의 소개
-> section1/1.강의소개.md
```

강의 제목이 명확하지 않으면 스크립트의 핵심 내용을 보고 짧은 제목을 새로 붙입니다.

## 문서 작성 원칙

- 원문을 그대로 붙여넣지 않습니다.
- 강의의 의도와 학습 포인트가 드러나도록 재구성합니다.
- 한 문단은 짧게 유지해 복습하기 쉽게 만듭니다.
- React Native, React, 컴포넌트 같은 핵심 용어는 필요한 경우 영어 원문을 함께 적습니다.
- 코드나 명령어가 등장하면 별도의 코드 블록으로 분리합니다.
- 아직 배우지 않은 개념을 과하게 확장해 설명하지 않습니다.

## 기본 노트 형식

```markdown
# M. 강의 제목

## 핵심 요약

- 강의의 핵심 내용을 3-5개 bullet로 정리합니다.

## 학습 목표

- 이 강의를 듣고 이해해야 할 내용을 정리합니다.

## 주요 내용

### 소제목

강의 내용을 주제별로 나누어 설명합니다.

## 정리

이번 강의에서 기억해야 할 내용을 짧게 마무리합니다.
```

## 예시

입력:

```text
1-1 안녕하세요 강의에 오신 걸 환영합니다
...
React Native란 무엇일까요
```

출력:

```text
section1/1.강의소개.md
```

예시 결과물은 [section1/1.강의소개.md](./section1/1.%EA%B0%95%EC%9D%98%EC%86%8C%EA%B0%9C.md)에 있습니다.
