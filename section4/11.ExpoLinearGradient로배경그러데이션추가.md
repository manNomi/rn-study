# 11. Expo LinearGradient로 배경 그러데이션 추가

## 핵심 요약

- 단일 배경색 대신 선형 그러데이션을 적용하기 위해 `expo-linear-gradient` 패키지를 사용한다.
- Expo 프로젝트에서는 일반 `npm install`보다 `expo install`을 사용하면 현재 Expo 버전에 맞는 의존성 버전을 설치할 수 있다.
- `LinearGradient` 컴포넌트를 import한 뒤 기존 루트 `View` 대신 사용한다.
- `LinearGradient`도 `style` prop을 지원하므로 기존 `flex: 1` 스타일을 그대로 적용할 수 있다.
- 그러데이션 색상은 `colors` prop에 배열로 전달한다.

## 학습 목표

- Expo 패키지를 설치할 때 `expo install`을 사용하는 이유를 이해한다.
- `expo-linear-gradient`의 `LinearGradient` 컴포넌트 사용법을 익힌다.
- 루트 배경을 단일 색상에서 그러데이션으로 바꾸는 흐름을 배운다.
- `colors` 배열로 그러데이션 색상을 구성하는 방법을 이해한다.
- 배경과 카드 UI가 충분히 구분되도록 색상 대비를 조정해야 한다는 점을 익힌다.

## 주요 내용

### 단일 배경색에서 그러데이션으로

이전 강의에서는 `App.js`에서 루트 `View`를 만들고 `backgroundColor`를 적용했다.

```jsx
const styles = StyleSheet.create({
  rootScreen: {
    flex: 1,
    backgroundColor: '#ddb52f',
  },
});
```

이 방식은 단순한 배경색을 만들기에는 충분하다.

하지만 이번에는 배경을 더 풍부하게 만들기 위해 선형 그러데이션을 적용한다.

최종적으로는 다음 순서로 배경을 발전시킨다.

```text
1. 단일 배경색
2. 그러데이션 배경
3. 배경 이미지 + 그러데이션
```

이번 강의는 두 번째 단계인 그러데이션 배경을 추가하는 단계다.

### expo-linear-gradient 패키지 설치

Expo는 React Native 개발을 쉽게 해주는 여러 패키지를 제공한다.

선형 그러데이션을 적용하려면 `expo-linear-gradient` 패키지를 사용한다.

강의에서는 터미널의 새 탭을 열어 다음 명령어로 설치한다.

```bash
expo install expo-linear-gradient
```

Expo 프로젝트에서는 일반적인 `npm install` 대신 `expo install`을 사용하는 것이 좋다.

이유는 `expo install`이 현재 프로젝트에서 사용하는 Expo SDK 버전을 확인한 뒤, 그 버전과 호환되는 패키지 버전을 설치해주기 때문이다.

```text
npm install
-> 패키지의 최신 버전 또는 지정 버전 설치

expo install
-> 현재 Expo SDK와 잘 맞는 버전 설치
```

즉 Expo 생태계에 포함된 패키지는 `expo install`로 설치하면 버전 충돌 가능성을 줄일 수 있다.

프로젝트 환경에 따라 다음처럼 실행할 수도 있다.

```bash
npx expo install expo-linear-gradient
```

설치가 끝난 뒤에도 기존 `npm start` 프로세스는 계속 실행해두면 된다.

### LinearGradient import하기

패키지를 설치한 뒤 `App.js`에서 `LinearGradient` 컴포넌트를 import한다.

```jsx
import { LinearGradient } from 'expo-linear-gradient';
```

기존에는 `View`를 사용해 루트 배경을 만들었다.

```jsx
import { StyleSheet, View } from 'react-native';
```

하지만 이제 루트 컨테이너로 `LinearGradient`를 사용할 것이므로, 더 이상 `View`를 쓰지 않는다면 `View` import는 제거할 수 있다.

```jsx
import { StyleSheet } from 'react-native';
import { LinearGradient } from 'expo-linear-gradient';
```

### View 대신 LinearGradient 사용하기

기존 루트 구조는 다음과 같았다.

```jsx
export default function App() {
  return (
    <View style={styles.rootScreen}>
      <StartGameScreen />
    </View>
  );
}
```

이제 `View`를 `LinearGradient`로 바꾼다.

```jsx
export default function App() {
  return (
    <LinearGradient style={styles.rootScreen}>
      <StartGameScreen />
    </LinearGradient>
  );
}
```

`LinearGradient`도 컴포넌트이므로 자식 컴포넌트를 감쌀 수 있다.

또한 `style` prop을 지원하기 때문에 기존 `rootScreen` 스타일을 그대로 적용할 수 있다.

### flex: 1은 그대로 유지하기

루트 배경이 화면 전체를 덮으려면 여전히 `flex: 1`이 필요하다.

```jsx
const styles = StyleSheet.create({
  rootScreen: {
    flex: 1,
  },
});
```

이전 강의에서 배운 것처럼, 루트 컨테이너가 화면 전체를 차지하지 않으면 배경도 일부 영역에만 적용된다.

`LinearGradient` 역시 마찬가지다.

```text
LinearGradient에 flex: 1 없음
-> 자식 콘텐츠 높이만큼만 그러데이션 표시

LinearGradient에 flex: 1 있음
-> 화면 전체에 그러데이션 표시
```

그러데이션을 사용할 때는 `backgroundColor`가 더 이상 필요하지 않다.

그러데이션 자체가 여러 색상을 배경으로 그리기 때문이다.

### colors prop으로 그러데이션 색상 설정하기

`LinearGradient`의 핵심 prop은 `colors`다.

`colors`에는 그러데이션에 사용할 색상을 배열로 전달한다.

```jsx
<LinearGradient
  colors={['#4e0329', '#ddb52f']}
  style={styles.rootScreen}
>
  <StartGameScreen />
</LinearGradient>
```

배열에는 두 개 이상의 색상을 넣을 수 있다.

```jsx
colors={['#4e0329', '#ddb52f']}
```

```jsx
colors={['#4e0329', '#72063c', '#ddb52f']}
```

색상이 두 개라면 첫 번째 색상에서 두 번째 색상으로 자연스럽게 변한다.

색상을 더 많이 넣으면 여러 단계의 그러데이션을 만들 수 있다.

### 전체 App.js 예시

이번 단계의 `App.js`는 다음과 같은 형태가 된다.

```jsx
import { StyleSheet } from 'react-native';
import { LinearGradient } from 'expo-linear-gradient';

import StartGameScreen from './screens/StartGameScreen';

export default function App() {
  return (
    <LinearGradient
      colors={['#4e0329', '#ddb52f']}
      style={styles.rootScreen}
    >
      <StartGameScreen />
    </LinearGradient>
  );
}

const styles = StyleSheet.create({
  rootScreen: {
    flex: 1,
  },
});
```

이렇게 하면 앱의 루트 배경이 단일 색상이 아니라 자두색에서 노란색으로 이어지는 그러데이션이 된다.

### 배경과 입력 카드의 대비 조정

그러데이션을 추가한 뒤에는 입력 카드가 배경과 잘 구분되는지 확인해야 한다.

배경과 카드 색이 너무 비슷하면 UI가 흐릿해 보일 수 있다.

강의에서는 입력 카드 배경색을 더 어두운 자두색으로 조정할 수 있다고 설명한다.

```jsx
inputContainer: {
  backgroundColor: '#3b021f',
}
```

이렇게 하면 입력 카드가 그러데이션 배경 위에서 더 선명하게 보인다.

```text
배경
-> 자두색과 노란색의 그러데이션

입력 카드
-> 더 어두운 자두색

결과
-> 카드와 배경의 시각적 구분이 강화됨
```

배경을 꾸밀 때는 단순히 예쁜 색을 고르는 것보다, 실제 UI 요소가 충분히 읽히고 구분되는지를 함께 봐야 한다.

### Expo 패키지를 활용하는 장점

`expo-linear-gradient`를 사용하면 직접 네이티브 코드를 작성하지 않고도 선형 그러데이션을 쉽게 적용할 수 있다.

React Native에서 모든 기능을 직접 구현하려고 하면 플랫폼별 네이티브 설정이 복잡해질 수 있다.

Expo는 이런 흔한 기능들을 패키지 형태로 제공해준다.

```text
선형 그러데이션
-> expo-linear-gradient

이미지 선택
-> expo-image-picker

폰트 로딩
-> expo-font
```

이번 강의에서 중요한 점은 Expo 패키지를 활용하면 React Native 앱의 시각적 기능을 비교적 간단하게 확장할 수 있다는 것이다.

### 프론트엔드 개발자 관점에서의 핵심 차이

웹에서는 CSS로 다음처럼 배경 그러데이션을 쉽게 줄 수 있다.

```css
body {
  min-height: 100vh;
  background: linear-gradient(#4e0329, #ddb52f);
}
```

React Native에는 CSS의 `linear-gradient()`가 그대로 존재하지 않는다.

대신 그러데이션을 그려주는 별도 컴포넌트를 사용한다.

```text
웹
-> CSS background: linear-gradient(...)

React Native + Expo
-> <LinearGradient colors={[...]} />
```

즉 React Native에서는 시각 효과도 스타일 프로퍼티 하나로 끝나지 않고, 별도 컴포넌트를 배치해 구현하는 경우가 많다.

이때도 핵심은 동일하다.

```text
화면 전체에 적용하려면
-> 루트 컨테이너로 사용
-> style에 flex: 1 유지
-> colors prop으로 색상 배열 전달
```

## 정리

이번 강의에서는 Expo가 제공하는 `expo-linear-gradient` 패키지를 설치하고, `LinearGradient` 컴포넌트로 앱 루트 배경에 선형 그러데이션을 적용했다. Expo 프로젝트에서는 `expo install`을 사용하면 현재 Expo SDK와 호환되는 패키지 버전을 설치할 수 있다.

기존 루트 `View`를 `LinearGradient`로 바꾸고, `style={styles.rootScreen}`을 유지해 `flex: 1`로 화면 전체를 채우도록 했다. 그러데이션 색상은 `colors` 배열로 전달하며, 배경을 적용한 뒤에는 입력 카드와 배경 사이의 대비도 함께 확인해야 한다.
